## Basic features

Text with **bold**, *italic*, `monospace`, <ins>underline</ins> and ~~strikethrough~~.

Absolute link to an [external location](http://example.com/) with a [query string](<http://example.com/?key[]=value>).

Relative link to a [heading in the same document](#Basic-features).

"Naked" links: <https://example.com> and <mailto:me@example.com>.

An ordered list:

1. *List item 1*
2. **List item 2**
spanning multiple text file lines but rendered as a single line
3. _List item 3_
4. `List item 4`

A bullet point list:

* *List item 1*
* **List item 2**
spanning multiple text file lines but rendered as a single line
* _List item 3_
* `List item 4`

An unordered list:

- *List item 1*
- **List item 2**
spanning multiple text file lines but rendered as a single line
- _List item 3_
- `List item 4`

A nested list:

* List item 1
    * Nested item 1
        1. Item 1
        2. Item 2
        3. Item 3
    * Nested item 2
        - Item 4
        - Item 5
        - Item 6
* List item 2
    1. Nested item 3
    2. Nested item 4
